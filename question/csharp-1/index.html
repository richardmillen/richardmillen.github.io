C# Quiz

1

What are the differences between structs and classes?


Unlike in C++, the default access level for both is private.
Their memory allocation models are different. A struct is a value type and its instances are created on the stack. A class is a reference type and its instances are created on the heap.

What is the best way to implement a thread-safe singleton?


public class Singleton {
    Singleton() {
    }
    public static Singleton Instance {
    get {
         return Nested.Instance;
    }
    class Nested {
         static Nested() {
         }
         internal static readonly Singleton instance = new Singleton();
    }
}

How does the ProgId attribute affect COM interop class registration?


The value of the progid attribute determines the registry key name in HKCR.
The namespace/class name combination determines the default value of said key.
Late binding (CreateObject etc.) uses the key name.
Early binding (New’ing etc.) uses the default value of the key.

4

When and how is a finalizer invoked?


It’s invoked by the garbage collector, the time is unknown to the programmer.

5

What is a static constructor?


It is invoked automatically when its type is executed by the runtime for the first time and is only invoked once.
