CentOS Quiz

1

Which configuration file does the OpenSSH daemon use?


/etc/ssh/sshd_config

2

What command is used to start the OpenSSH service?


/sbin/service sshd start

3

What command is used to stop the OpenSSH service?


/sbin/service sshd stop

4

If SSH is used, which insecure services should be disabled?


  • telnet
  • rsh
  • rlogin
  • vsftpd

5

What is the default system-wide SSH client configuration file?
How may it be overridden?


/etc/ssh/ssh_config
~/.ssh/config

6

Where are the authorised public keys of SSH clients held?


~/.ssh/authorized_keys

7

How is it possible to transfer a local file to a remote system over a secure, encrypted connection?


scp <local-file> <user>@<server>:<remote-file>

8

How is it possible to transfer a remote file to the local system over a secure, encrypted connection?


scp <user>@<server>:<remote-file> <local-file>

9

How is it possible to transfer the contents of a local directory to a remote system over a secure, encrypted connection?


scp <local-path>/* <user>@<server>:<remote-path>/

10

What command do you use to generate an RSA key pair for SSH?


ssh-keygen -t rsa

11

What command do you use to generate a DSA key pair for SSH?


ssh-keygen -t dsa

12

What user setups must be done to allow access to a remote system over SSH?


chmod 755 ~/.ssh
cat ~/.ssh/<key-file>.pub >> ~/.ssh/authorized_keys
chmod 644 ~/.ssh/authorized_keys

13

How do you get the local ip address of a machine?


There are a few options:

hostname -I
ip addr
ifconfig

14

How do you find out which packages will provide a feature or file?


yum provides <feature>
