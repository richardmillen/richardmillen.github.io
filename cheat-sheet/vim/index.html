VIM Cheat Sheet

Cursor

Move Jump Other
h - move cursor left w - jump forwards to the start of a word gg - go to the first line of the document
j - move cursor down W - jump forwards to the start of a word (words can contain punctuation) G - go to the last line of the document
k - move cursor up e - jump forwards to the end of a word 5G - go to line 5
l - move cursor right E - jump forwards to the end of a word (words can contain punctuation) ; - repeat previous f, t, F or T movement
H - move to top of screen b - jump backwards to the start of a word , - repeat previous f, t, F or T movement, backwards
M - move to middle of screen B - jump backwards to the start of a word (words can contain punctuation) zz - center cursor on screen
L - move to bottom of screen % - move to matching character (default supported pairs: ‘()’, ‘{}’, ‘[]’ - use :h matchpairs in vim for more info) Ctrl + b - move back one full screen
0 - jump to the start of the line Ctrl + f - move forward one full screen
^ - jump to the first non-blank character of the line Ctrl + d - move forward 1⁄2 a screen
$ - jump to the end of the line Ctrl + u - move back 1⁄2 a screen
g_ - jump to the last non-blank character of the line
fx - jump to next occurrence of character x
tx - jump to before next occurrence of character x
Fx - jump to previous occurence of character x
Tx - jump to after previous occurence of character x
} - jump to next paragraph (or function/block, when editing code)
{ - jump to previous paragraph (or function/block, when editing code)

Insert Mode

Inserting / Appending Text Editing Cut & Paste
i - insert before the cursor r - replace a single character yy - yank (copy) a line
I - insert at the beginning of the line J - join line below to the current one with one space in between 2yy - yank (copy) 2 lines
a - insert (append) after the cursor gJ - join line below to the current one without space in between yw - yank (copy) the characters of the word from the cursor position to the start of the next word
A - insert (append) at the end of the line cc - change (replace) entire line y$ - yank (copy) to end of line
o - append (open) a new line below the current line cw - change (replace) to the end of the word p - put (paste) the clipboard after cursor
O - append (open) a new line above the current line c$ - change (replace) to the end of the line P - put (paste) before cursor
ea - insert (append) at the end of the word s - delete character and substitute text dd - delete (cut) a line
Esc - exit insert mode S - delete line and substitute text (same as cc) 2dd - delete (cut) 2 lines
xp - transpose two letters (delete and paste) dw - delete (cut) the characters of the word from the cursor position to the start of the next word
u - undo D - delete (cut) to the end of the line
Ctrl + r - redo d$ - delete (cut) to the end of the line
. - repeat last command x - delete (cut) character

Visual Mode

Marking Text Visual Commands
v - start visual mode, mark lines, then do a command (like y-yank) > - shift text right
V - start linewise visual mode < - shift text left
o - move to other end of marked area y - yank (copy) marked text
Ctrl + v - start visual block mode d - delete marked text
O - move to other corner of block ~ - switch case
aw - mark a word
ab - a block with ()
aB - a block with {}
ib - inner block with ()
iB - inner block with {}
Esc - exit visual mode

Search

Search & Replace Search in multiple files
/pattern - search for pattern :vimgrep /pattern/ {file} - search for pattern in multiple files e.g. :vimgrep /foo/ **/*
?pattern - search backward for pattern :cn - jump to the next match
\vpattern - ‘very magic’ pattern: non-alphanumeric characters are interpreted as special regex symbols (no escaping needed) :cp - jump to the previous match
n - repeat search in same direction :copen - open a window containing the list of matches
N - repeat search in opposite direction
:%s/old/new/g - replace all old with new throughout file
:%s/old/new/gc - replace all old with new throughout file with confirmations
:noh - remove highlighting of search matches

Multiple Files

Buffers Windows Tabs
:e file - edit a file in a new buffer :sp file - open a file in a new buffer and split window :tabmove # - move current tab to the #th position (indexed from 0)
:bn or :bnext - go to the next buffer :vsp file - open a file in a new buffer and vertically split window Ctrl + wT - move the current split window into its own tab
:bp or :bprev - go to the previous buffer Ctrl + ws - split window gt or :tabnext or :tabn - move to the next tab
:b4 - go to buffer number 4 Ctrl + ww - switch windows gT or :tabprev or :tabp - move to the previous tab
:bf - go to first buffer Ctrl + wq - quit a window #gt - move to tab number #
:bl - go to last buffer Ctrl + wv - split window vertically :tabclose or :tabc - close the current tab and all its windows
:b foo<TAB> - go to buffer foo (with tab completion) Ctrl + wh - move cursor to the left window (vertical split) :tabonly or :tabo - close all tabs except for the current one
:bd - delete a buffer (close a file) Ctrl + wl - move cursor to the right window (vertical split) :tabdo command - run the command on all tabs (e.g. :tabdo q - closes all opened tabs)
:ls - list all open buffers Ctrl + wj - move cursor to the window below (horizontal split) :tabnew or :tabnew file - open a file in a new tab
Ctrl + wk - move cursor to the window above (horizontal split)

Productivity

Registers Marks Macros
:reg - show registers content :marks - list of marks qa - record macro a
"xy - yank into register x ma - set current position for mark A q - stop recording macro
"xp - paste contents of register x `a - jump to position of mark A @a - run macro a
y`a - yank text to position of mark A @@ - rerun last run macro

Application

Global Exiting
:help keyword - open help for keyword :w - write (save) the file, but don’t exit
:o file - open file :w !sudo tee % - write out the current file using sudo
:saveas file - save file as :wq or :x or ZZ - write (save) and quit
:close - close current pane :q - quit (fails if there are unsaved changes)
K - open man page for word under the cursor :q! or ZQ - quit and throw away unsaved changes
:wqa - write (save) and quit on all tabs

A VIM Workflow

  • Open VIM, use edit (with regex) to open buffer.
  • Use edit to open another file, OR use sp / vsp if you want to toggle.
  • Repeat, using techniques below to switch between files
  • Close and re-open VIM whenever necessary.

Techniques

  • Use buffers: :b
  • Jump / change lists:
    • Ctrl + o
    • Ctrl + i
    • g;
  • Alternate file (^)

Tips & Tricks

Switching between files

Opening files
:e **/*foo<TAB> - choose from all the files with foo in their name under pwd
:e **/*foo/*bar<TAB> - choose from all files containing bar in their name under any subdir under pwd containing foo
:tabe[dit] ... - same as edit
:sp[lit] ... - same as edit
:vs[plit] ... - same as edit
:next **/*.js - open all files under pwd with a .js extension (TODO: :help arglist)
