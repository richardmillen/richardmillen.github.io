VIM Cheat Sheet

Multiple Files

Buffers

Command Info
:w write (save) changes to file
:wa write all changed files
:w !sudo tee % write out the current file using sudo (use case?)
:e foo or :edit foo edit a file in a new buffer
:r foo or :read foo or :read !foo read a file or output of shell command
:10read foo read and insert file / shell output after line 10
:bn or :bnext switch to the next buffer
:bp or :bprev switch to the previous buffer
:b4 switch to buffer number 4
:bf switch to first buffer
:bl switch to last buffer
:b foo<TAB> switch to buffer foo
:bd delete a buffer (close a file)
:ls or :buffers list all open buffers
:set autoread configure to reload stale, out-of-date buffers from disk; bypass confirm message when :checkt
:checktime force reload out-of-date buffers from disk
set hidden hide buffers when abandoned
set splitbelow, set splitright new split panes at bottom, right

Windows

Open & Close

Command Info
:sb foo<TAB> split window and edit file / buffer
:sp or :sp foo split window in two or edit file in new split window
:10sp foo new split 10 lines tall
:vs foo<TAB> open a file in a new buffer and vertically split window
:vert sb N open buffer #N in left vertical split
:vert belowright sb N open buffer #N in right vertical split
CTRL-W T turn window into new tab
CTRL-W o close all windows but the current (in the current tab)
CTRL-W q quit a window

Configure

Command Info
CTRL-W = normalise split window sizes
CTRL-W R swap top/bottom or left/right split
CTRL-W + or - inc or dec height; add w + N for multiple lines
CTRL-W > or < inc or dec width; add w + N for multiple columns
set number, set nonumber show, hide line numbers
set nu! toggle line numbers

Navigate

Command Info
CTRL-W w switch windows
CTRL-W h move cursor to the left window (vertical split)
CTRL-W l move cursor to the right window (vertical split)
CTRL-W j move cursor to the window below (horizontal split)
CTRL-W k move cursor to the window above (horizontal split)

Tabs

Command Info
:tabmove # move current tab to the #th position (indexed from 0)
gt or :tabnext or :tabn move to the next tab
gT or :tabprev or :tabp move to the previous tab
#gt move to tab number #
:tabclose or :tabc close the current tab and all its windows
:tabonly or :tabo close all tabs except for the current one
:tabdo command run the command on all tabs (eg :tabdo q
:tabnew or :tabnew file open a file in a new tab

Find Text

Search

Command Info
* search for next occurrence of current word (under cursor)
# search for previous occurrence of current word (under cursor)
/pattern search for pattern
?pattern search backward for pattern
n repeat search in same direction
N repeat search in opposite direction
\vpattern ‘very magic’ pattern: non-alphanumeric characters are interpreted as special regex symbols (no escaping needed)

Substitute

Command Info
:%s/CTRL-R0/new/g replace all occurrences of previously yanked (copied) text with new; entire file
:s/\<old\>/new/g replace all old (exact matches) with new; current line only
5,12s/old/new/g replace all old with new from line 5 to 12 inclusive
:%s/old/new/g, c, i replace all old with new throughout file, with confirmation, ignoring case
:.,$s/old/new/g replace all old with new from current line to last line inclusive
:.,+2s/foo/bar/g replace all old with new for current line and next two lines

Global Operation

Command Info
:g/foo/d delete all occurrences of foo
:g/^foo/s/old/new/g replace each old with new in each line starting with foo

Configure

Command Info
:set hlsearch or :set nohlsearch do or do not highlight all future search matches
:noh turn off current highlighting (until next search)
:set hlsearch! toggle searching highlighting
:set ignorecase or :set noignorecase ignore or don’t ignore case when searching

Search in multiple files

Command Info
:vimgrep /pattern/ {file} search for pattern in multiple files eg :vimgrep /foo/ **/*
:cn jump to the next match
:cp jump to the previous match
:copen open a window containing the list of matches

Cursor

Move

Command Info
h move cursor left
j move cursor down
k move cursor up
l move cursor right
H move to top of screen
M move to middle of screen
L move to bottom of screen

Return

Command Info
`` return to where last jump from (two backticks!)
CTRL-O retrace your movements from newest to oldest along the jump list
CTRL-I retrace your movements from oldest to newest along the jump list (after CTRL-O)

Characters

Command Info
fx jump to next x on the line
Fx jump to previous x on the line
tx jump to before next x on the line
Tx jump to after previous x on the line
; repeat previous f, t, F or T movement
, repeat previous f, t, F or T movement (opposite direction)
^ jump to the first non-blank character of the line
g_ jump to the last non-blank character of the line

Words

Command Info
w jump forward to the start of a word
W jump forward to the start of a word (words can contain punctuation)
e jump forward to the end of a word
E jump forward to the end of a word (words can contain punctuation)
b jump back to the start of a word
B jump back to the start of a word (words can contain punctuation)
} jump to next paragraph (or function/block, when editing code)
{ jump to previous paragraph (or function/block, when editing code)

Sentences

Command Info
) jump forward one sentence
( jump back one sentence

Blocks

Command Info
% jump to matching char: ‘()’, ‘{}’, ‘[]’ (:help matchpairs)
[{ jump back to opening brace
]} jump forward to closing brace
[( jump to opening paren
]) jump to closing paren
[[ jump back to previous { (first column)
]] jump forward to next { (first column)
[] jump back to previous } (first column)

Lines

Command Info
0 jump to the start of the line
$ jump to the end of the line
gg go to the first line of the document
G go to the last line of the document
5G go to line 5

Screen

Command Info
zz center cursor on screen
CTRL-B move back one full screen
CTRL-F move forward one full screen
CTRL-D move forward 1⁄2 a screen
CTRL-U move back 1⁄2 a screen

Insert Mode

Entering

Command Info
c change; delete current visual selection and insert
ciw change; delete (inner) word and insert
ci," change; delete (inner) content inside " and insert; supports all different quotes and parens
i insert before the cursor
I insert at the beginning of the line
a insert (append) after the cursor
A insert (append) at the end of the line
o append (open) a new line below the current line
O append (open) a new line above the current line
ea insert (append) at the end of the word
ESC exit insert mode

Visual Mode

General

Command Info
. repeat last command

Replace

Command Info
cc change (replace) entire line
cw change (replace) to the end of the word
c$ change (replace) to the end of the line
r replace a single character

Editing

Command Info
u undo
CTRL-R redo
J join line below to the current one with one space in between
gJ join line below to the current one without space in between
s delete character and substitute text
S delete line and substitute text (same as cc)
xp transpose two letters (delete and paste)
> shift text right
< shift text left
d delete marked text
~ switch case

Blocks

Command Info
CTRL-V start visual block mode
SHIFT-I insert before each line in block, then ESC to finish
SHIFT-A append after each line in block, then ESC to finish

Copy

Command Info
y yank (copy) marked text
yy yank (copy) a line
2yy yank (copy) 2 lines
yw yank (copy) the characters of the word from the cursor position to the start of the next word
y$ yank (copy) to end of line

Cut

Command Info
dd delete (cut) a line
2dd delete (cut) 2 lines
dw delete (cut) the characters of the word from the cursor position to the start of the next word
D delete (cut) to the end of the line
d$ delete (cut) to the end of the line
x delete (cut) character

Paste

Command Info
p put (paste) the clipboard after cursor
P put (paste) before cursor
viwp select inner word and paste (overwrite)
vep select to end of word and paste (overwrite)
:pu put (paste) after current line
:pu! 10 put (paste) before line 10

Marking Text

Command Info
v start visual mode, mark lines, then do a command (like y-yank)
V start linewise visual mode
o move to other end of marked area
O move to other corner of block
aw mark a word
ab a block with ()
aB a block with {}
ib inner block with ()
iB inner block with {}
Esc exit visual mode

Productivity

Registers

Command Info
:reg show registers content
"xy yank into register x
"xp paste contents of register x

Marks

Command Info
:marks list of marks
ma set current position for mark A
`a - jump to position of mark A
y`a - yank text to position of mark A

Macros

Command Info
qa record macro a
q stop recording macro
@a run macro a
@@ rerun last run macro

Application

Global

Command Info
:help keyword open help for keyword
:o file open file
:saveas file save file as
:close close current pane
K open man page for word under the cursor

Exiting

Command Info
:wq or :x or ZZ write (save) and quit
:q quit (fails if there are unsaved changes)
:q! or ZQ quit and throw away unsaved changes
:wqa write (save) and quit on all tabs

A VIM Workflow

  • Open VIM, use edit (with regex) to open buffer
  • Use edit to open another file, OR use sp / vsp if you want to toggle
  • Repeat, using techniques below to switch between files
  • Close and re-open VIM whenever necessary

Techniques

  • Use buffers: :b
  • Jump / change lists:
    • CTRL-O
    • CTRL-I
    • g;
  • Alternate file (^)
