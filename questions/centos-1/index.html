CentOS Quiz

Below are some short quizes about the CentOS operation system.

Quiz 1

  1. Which configuration file does the OpenSSH daemon use?
  2. What command is used to start the OpenSSH service?
  3. What command is used to stop the OpenSSH service?
  4. If SSH is used, which insecure services should be disabled?
  5. What is the default system-wide SSH client configuration file? How may it be overriden?

1.

/etc/ssh/sshd_config

2.

/sbin/service sshd start

3.

/sbin/service sshd stop

4.

  • telnet
  • rsh
  • rlogin
  • vsftpd

5.

/etc/ssh/ssh_config
~/.ssh/config

Quiz 2

  1. Where are the authorised public keys of SSH clients held?
  2. How is it possible to transfer a local file to a remote system over a secure, encrypted connection?
  3. How is it possible to transfer a remote file to the local system over a secure, encrypted connection?
  4. How is it possible to transfer the contents of a local directory to a remote system over a secure, encrypted connection?
  5. What command do you use to generate an RSA key pair for SSH?

1.

~/.ssh/authorized_keys

2.

scp <local-file> <user>@<server>:<remote-file>

3.

scp <user>@<server>:<remote-file> <local-file>

4.

scp <local-path>/* <user>@<server>:<remote-path>/

5.

ssh-keygen -t rsa

Quiz 3

  1. What command do you use to generate a DSA key pair for SSH?
  2. What user setups must be done to allow access to a remote system over SSH?
  3. How do you get the local ip address of a machine?
  4. How do you find out which packages will provide a feature or file?

1.

ssh-keygen -t dsa

2.

chmod 755 ~/.ssh
cat ~/.ssh/<key-file>.pub >> ~/.ssh/authorized_keys
chmod 644 ~/.ssh/authorized_keys

3.

hostname -I

OR

ip addr

OR

ifconfig

4.

yum provides <feature>
