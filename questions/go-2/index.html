1.

OR

2.

When a new slice is created from a portion of the underlying array of an existing slice.
Both slices now point to the same underlying array.
New slice taken from start index, up to, but not including an end index.
New slice length = end - start.
New slice capacity = capacity - start.

newslice := myslice[start:end]

3.

To append a value to a slice, growing its capacity if needed.
It returns a slice with the newly appended value.

newslice = append(myslice, myvalue)

4.

Length of B is increased to 4.
Capacity of B remains the same.

5.

The values of the existing underlying array are copied to a new underlying array.
If existing capacity < 1,000, capacity is doubled.
If >= 1,000, capacity is grown by 25%.
